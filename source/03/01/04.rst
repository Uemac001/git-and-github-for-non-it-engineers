#####################################################################
オススメの拡張機能
#####################################################################

VSCodeで作業をする際に入れておくと便利な拡張機能を紹介します。

インストール方法は前項の日本語の際と同様に拡張機能タブで名前を検索してインストールするだけです。


*********************************************************************
Git History
*********************************************************************

Git History: https://marketplace.visualstudio.com/items?itemName=donjayamanne.githistory

以下のような機能があります。

- 変更履歴と検索

    - 単一のブランチまたは全てのブランチのコミット履歴
    - ファイル単位の変更履歴

- 比較

    - ブランチの比較
    - コミットの比較
    - コミット間でのファイルの比較

- Gitの操作

    - ブランチのマージ
    - ブランチの作成
    - 任意のコミットの状態にリセット
